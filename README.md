# Gerador
##### Gerador automatico de arquivos
##### Cria toda estrutura de diretórios de uma aplicação web básica

# Intalando as dependências
#####pip -r requeriments.txt

# Dependências
* wget => 3.2

# Como usar
##### ./gerador.py [arg] [arg]
##### ./gerador.py [tipo do arquivo] [nome do projeto]

# Exemplo
##### ./gerador.py html app
